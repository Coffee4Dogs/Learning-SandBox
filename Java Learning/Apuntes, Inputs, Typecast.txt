Que es un arreglo?
Un arreglo es una variable que puede guardar mas de un valor secuencialmente y del mismo tipo.

Areglos bidimensionales. ----------------------------------
Es una variable que puede almacenar un conjunto de datos organizados en filas y columnas.
Objetivo: Comprender como se organizan y se organizan, se almacenan y se recorren los datos en un arreglo bidimencional.


x = A.length; = cuantas filas tiene el array.
x = A[1].length; = cuantas colunas tiene el array.
