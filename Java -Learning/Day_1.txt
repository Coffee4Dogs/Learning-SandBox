Datos simples cuando son datos puntuales. Como Juan, 57, true, etc.
Datos estructurados, cuando estan compuestos por un conjunto de datos de igual o diferente tipo.

Los datos numericos se dividen en:
  - Int (Enteros, que no tienen punto decimal.)
  - Float (Flotante o Double) 4 bytes.
  - String (Alfanumericos, Letras, Digitos, etc)
  - Bool (Booleanos, True/False)
  - Double es como un flotante pero de doble precision.


Partes de un programa:
- Declaracion de variables.
- Entrada de datos
- Proceso de datos
- Salida de datos
